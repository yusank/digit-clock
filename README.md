# digitalClock
simple terminal digital clock implement by go .

## how to use

``` shell
$ go get github.com/yusank/digitalClock

$ digitalClock

$ digitalClock -s
```

or

``` shell
$ git clone github.com/yusank/digitalClock
$ cd digitalClock
$ go build
$ digitalClock -s
```

> `-s` for show the second
